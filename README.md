# Windows-Server-Ad-Lab
This repository documents my Windows Server 2022 Home Lab, created to develop the practical skills used by IT Administrators, System Administrators, Help Desk Technicians, and Cybersecurity professionals.

The goal of this project is to simulate a small enterprise network using Windows Server 2022 and Windows 11 Enterprise virtual machines. Rather than simply following tutorials, every service is configured, tested, broken, troubleshot, and documented to mirror real-world IT administration tasks.

Throughout this lab I deploy and manage Active Directory, Group Policy, DNS, DHCP, user and computer accounts, organizational units, security groups, shared folders, NTFS permissions, PowerShell automation, and common troubleshooting scenarios. Every phase includes screenshots, configuration notes, troubleshooting documentation, and lessons learned to demonstrate both technical knowledge and problem-solving skills.

This lab serves as an ongoing portfolio project that showcases practical experience with Microsoft enterprise technologies commonly used in professional IT environments.
